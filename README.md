# TSLint Config

[TSLint config](https://palantir.github.io/tslint/) rules config

## Installation

```sh
npm i --save-dev @gnarlycode/tslint-config
```

## Usage

In `tslint.json`:

```json
{
  "extends": "@gnarlycode/tslint-config"
}
```
